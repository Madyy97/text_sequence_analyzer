# Text Sequence Analyzer

A simple text-analysis tool that reads a text file and reports word-frequency
statistics after basic NLP preprocessing.

## What it does
- Reads text from a file
- Cleans the text: removes punctuation, lowercases, removes common stopwords
- Counts word frequencies
- Reports total words, unique words, and the top 5 most frequent words
